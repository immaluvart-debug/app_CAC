

Leaf & Fork

An app that allows users to make smart ecological choices, raise awareness of green scores, and help users find similar food products with higher green scores.

This project is almost completed. There are a few features that need to tweaked to handle all user cases and more data sourcing for the near-me map

Features:
- Scan any food product and receive it's green score, nutritional information, and product information (if data available)
- Able to favorite products as well
- Able to see recommended food items similar to the product most recently scanned. You can filter out the green scores, nutritional information, calories, and more.
- Able to see locally produced food products near you [wip feature]
- AI Chatbot where you can ask any questions or recommendations about food choices
- Habit Tracker where you can input any items you scanned and their green scores on a daily, weekly, or monthly basis.
