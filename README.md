

Leaf & Fork

An app that allows users to make smart ecological choices, raise awareness of green scores, and help users find similar food products with higher green scores.

This project is almost completed. There are a few features that need to tweaked to handle all user cases and more data sourcing for the near-me map

Features:
- Scan any food product and receive it's green score, nutritional information, and product information (if data available)
- Able to favorite products as well
- Able to see recommended food items similar to the product most recently scanned. You can filter out the green scores, nutritional information, calories, and more.
- Able to see locally produced food products near you [wip feature]
- AI Chatbot where you can ask any questions or recommendations about food choices
- Habit Tracker where you can input any items you scanned and their green scores on a daily, weekly, or monthly basis.

HOW TO USE:
Use on iPhone and turn on Light Mode
1. Connect a computer (where the code is) to an iPhone via a cable so the app gets downloaded on the iPhone (You can also use simulator but scanning wont work with it)
2. Directly run the code and click "SIGN UP"
3. Enter in any email (ex: sanvi@gmail.com) and any password greater than 6 letters.
4. Choose preferences and click OK
5. Click "SCAN" to scan any item with the item's barcode.
6. Click "Product Info" to see available product information and "Nutrition Info" to see green score and nutrition data
7. Click "Recommended" once scanned to see recommended items similar to that product with better scores
8. Click the heart icon on the product to favorite it and go to the heart icon on the top right corner to see the favorites.
9. Click the calendar icon on top right to see the habit tracker. Click the + to add an entry!
10. Click the "NEAR ME" to see our Work In Progress feature that locates locally sourced food (no data yet, just fillers)
11. Swipe right on the home screen to see the AI Chatbot page. Ask it anything!
